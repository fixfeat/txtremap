# txt章节名格式化工具
txt chapter title reformatting tool

https://fixfeat.github.io/txtremap/

解决部分txt格式图书章节标题格式混乱，可能无法通过简单正则表达式解析章节标题的问题。
原理：找到文本中所有数字、中文数字的最长增长子序列。
无法解决的情况：章节顺序错误，或存在较大的序号跳跃。

使用方法：
1. 上传一个txt文件
2. 检查生成的目录，单击可以跳转到对应位置，双击切换是否包含在最终目录中。一般来讲，前二十章更有可能出现识别错误。
3. 下载生成的txt文件，最终标题行格式为：“第xx章 章节标题”（目前无配置项）

目前程序会直接忽略的情况（目前无配置项）：
1. 大于30字的行
2. 数字在本行第2个字符之后才出现
3. 数字是日期或时间的一部分
4. 数字偏离主序列过远（比上个主序列数字大300或小10）
5. 数字后面紧跟有正常不出现在章节序号后的特殊符号
6. 数字比文中所有数字数量还要大1000
